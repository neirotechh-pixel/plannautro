Как это работает

Три шага до чистого утра

Никаких сложных интерфейсов. Просто разговаривай с ботом в Telegram.

🎙️
01

Скажи или напиши

Отправь голосовое сообщение или текст с планами на день — в любой свободной форме, как другу.

🧠
02

ИИ разбивает на задачи

Whisper расшифрует речь. LLaMA 3.3 выделит задачи, найдёт время выполнения и составит список.

✅
03

Выполняй и отмечай

Задачи в Supabase. Отмечай выполненные через /done. Смотри прогресс через /today в любой момент.

Демо

Голос превращается
в задачи за секунды

Запись → Whisper транскрибирует → LLaMA разбивает на задачи → список готов. Всё происходит автоматически, пока ты пьёшь кофе.

Возможности

Технологии уровня про

В простом чат-боте — стек как у серьёзных продуктов.

🎙️

Голосовой ввод

Whisper large-v3 — лучшая модель распознавания речи. Точность 95%+ для русского языка, работает даже в шуме.

🧠

ИИ-разбивка задач

LLaMA 3.3 70B с температурой 0.3 извлекает задачи из произвольного текста. Автоматически находит время выполнения.

☁️

Облачное хранение

Задачи хранятся в Supabase Postgres с RLS-политиками. Доступны с любого устройства, данные изолированы по пользователям.

⚡️

Мгновенный ответ

Groq Inference — самый быстрый LLM-хостинг в мире. Ответ бота в 2–3 секунды на текст, 5–8 сек на голос.

Команды

Простой синтаксис

Четыре команды — всё что нужно.

/start
Начать работу и получить справку по боту
/today
Список всех задач на сегодня с прогрессом
/done 1
Отметить задачу выполненной по номеру
/clear
Очистить все задачи текущего дня
🌅

Начни утро правильно

Бесплатно. Без регистрации. Просто открой бота и скажи что планируешь сделать сегодня.
